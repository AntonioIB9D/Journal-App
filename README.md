# Journal-App 📄

Diario de notas ✨

La aplicación permite: 

1. Crear una nueva nota. ✏️
2. Editar una nota ya existente. 🟢
3. Eliminar una nota existente. ❌

---

### Notas adicionales 📗

##### Para utlizar la aplicación se deberan:

---

Instalar modulos de node ⬇️ :

```
npm install
```

Construir ejecutar ⚙️ :

```
npm run dev
```
